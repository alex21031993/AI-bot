# 🐋 Crypto Intelligence Agent

Комплексный AI-агент для анализа криптовалют, который анализирует социальные сети, активность китов, рыночные настроения и технические индикаторы для выявления перспективных активов.

## 📊 Возможности

### Анализ
- **Социальные сети** - анализ упоминаний в Twitter/X, Reddit, Telegram
- **Настроения рынка** - определение бычьих/медвежьих настроений
- **Отслеживание китов** - мониторинг крупных транзакций и кошельков
- **Технический анализ** - RSI, MACD, EMA, VWAP, Bollinger Bands
- **Анализ объемов** - ликвидность и объемы торгов
- **История монеты** - анализ пампов, дампов и событий

### AI Scoring
```
Social Score:      25%
Sentiment Score:   20%
Whale Score:       20%
Technical Score:   20%
Volume Score:      15%
─────────────────────────
```

### Риск-оценка
- **0-40%** - 🔴 Высокий риск
- **41-60%** - 🟠 Спекулятивный актив
- **61-80%** - 🟡 Перспективный актив
- **81-100%** - 🟢 Сильный сигнал

## 🚀 Быстрый старт

### Установка

```bash
# Клонируйте репозиторий
git clone https://github.com/alex21031993/AI-bot.git
cd AI-bot

# Установите зависимости
pip install -r requirements.txt
```

### Настройка

Создайте файл `.env` в корневой директории:

```env
# Telegram Bot Token (от @BotFather)
TELEGRAM_BOT_TOKEN=your_bot_token_here

# Admin User IDs (через запятую)
ADMIN_USER_IDS=123456789,987654321
```

### API Ключи (опционально)

| API | Стоимость | Что дает |
|-----|-----------|----------|
| **CoinGecko** | 🆓 Бесплатно | Рыночные данные, цены (уже работает без ключа) |
| **Reddit** | 🆓 Бесплатно | Анализ дискуссий (уже работает без ключа) |
| **Whale Alert** | 💰 Платно | Реальные данные о китах |
| **CoinMarketCap** | 💰 Платно | Расширенные рыночные данные |

### Запуск

